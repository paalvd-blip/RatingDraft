# Restaurantliste

## Før du laster opp til GitHub
Åpne `src/App.jsx` og fyll inn dine Supabase-nøkler øverst i filen:

```js
const SUPABASE_URL = "";
const SUPABASE_ANON_KEY = "";
```

## Deploy
1. Last opp hele denne mappen til et GitHub-repo
2. Gå til vercel.com → Add New → Project → velg repoet
3. Vercel oppdager Vite automatisk – bare trykk Deploy

## Lokal testing (valgfritt)
```
npm install
npm run dev
```
